svn-da(1)
=========

NAME
----
svn-da - Apply a given diff0 to a given source and produce fulltext

SYNOPSIS
--------
svn-da SOURCE <DIFF

DESCRIPTION
-----------
Applies an svndiff version 0 to a given source file.
