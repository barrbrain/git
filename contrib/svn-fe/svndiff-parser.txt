svndiff-parser(1)
=================

NAME
----
svn-fe - convert an SVN "dumpfile" to a fast-import stream

SYNOPSIS
--------
svndiff-parser SOURCE <DIFF

DESCRIPTION
-----------
Applies an svndiff version 0 to a given source file.
